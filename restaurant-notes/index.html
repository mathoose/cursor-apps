      How to use

      Save a spot from Instagram

      1. In Instagram, tap Share on the post → Copy link.
      2. Open Resto Notes (this app) → tap + Add.
      3. Tap Paste link next to Instagram. Add the name and anything else.
      4. Tap Done. It saves automatically.

      Move into Philly Dates

      1. Here: tap Copy all for import.
      2. Open Philly Dates → Settings (⚙️).
      3. Scroll to Import from text → paste → Import pasted text.

      Shortcuts (optional — skip if confusing)

      You don’t need Apple Shortcuts. Copy link + Paste link above is enough.
