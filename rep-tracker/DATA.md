# Rep Tracker data

Static files in `data/` are generated by `scripts/build-data.js`.

## Refresh

```bash
cd rep-tracker
node scripts/build-data.js
```

Options:

- `--skip-geo` — reuse existing `districts.geojson`
- `--skip-wiki` — skip Wikipedia election-margin fetch (faster; use cached JSON or CSV input)

## Sources

| Output | Source |
|--------|--------|
| `house.json`, `senate.json` members | [unitedstates/congress-legislators](https://unitedstates.github.io/congress-legislators/legislators-current.json) |
| Presidential margins (House districts) | [The Downballot](https://www.the-downballot.com/p/data) Google Sheets (via the-db.co/presbycd) |
| Presidential margins (states) | County aggregates from [tonmcg/US_County_Level_Election_Results_08-24](https://github.com/tonmcg/US_County_Level_Election_Results_08-24) |
| Party election margins | Wikipedia election pages (cached in `scripts/input/*-election-cache.json`) or optional CSV |
| Bolts key races | Hand-curated `scripts/input/bolts-watch.json` from [Bolts cheat sheets](https://boltsmag.org/whats-on-the-ballot/) |
| `districts.geojson` | [JeffreyBLewis/congressional-district-boundaries](https://github.com/JeffreyBLewis/congressional-district-boundaries) (119th Congress) |
| `states.topo.json` | [us-atlas](https://github.com/topojson/us-atlas) |

## Optional CSV overrides

Drop files in `scripts/input/` before building:

**`house-election-2024.csv`**

```csv
district,demPct,repPct
PA-07,49.5,50.5
```

**`senate-election-2024.csv`**

```csv
state,demPct,repPct
PA,51.2,48.8
```

Columns `partyMargin` may be used instead of `demPct`/`repPct`.
