# Rep Tracker data

Static files in `data/` are generated by `scripts/build-data.js`. Election history lives in committed `data/elections/` JSON.

## Refresh

```bash
cd rep-tracker
node scripts/build-data.js
```

Options:

- `--skip-geo` — reuse existing `districts.geojson` and `district-history/` if present
- `--skip-history` — reuse existing `district-history/` only

## District history (`data/district-history/`)

Historical boundary overlays for redrawn House districts (detail modal mini-map).

| File | Contents |
|------|----------|
| `index.json` | Per-district `changed` flag and state code (loaded at app init) |
| `{ST}.json` | Lazy-loaded shards: geometries + pres margins per vintage year |

Pres year → Congress boundary vintage:

| Pres year | Congress |
|-----------|----------|
| 2008 | 110th |
| 2012 | 113th |
| 2016 | 115th |
| 2020 | 117th |
| 2024 | 119th |

Only districts whose shape changed (centroid shift &gt; 2 km or bbox area ratio outside 0.85–1.15) are included in state shards.

## Election database (`data/elections/`)

| File | Contents |
|------|----------|
| `pres-by-cd.json` | Presidential margin (D−R) by congressional district, 2008–2024 |
| `pres-by-state.json` | Presidential margin by state, 2008–2024 |
| `house-results.json` | 2024 House general election results by district |
| `senate-results.json` | 2024 Senate general election results (Class 1 states) |
| `meta.json` | Version, build time, row counts |

## Sources (no Wikipedia)

| Output | Source |
|--------|--------|
| `house.json`, `senate.json` members | [unitedstates/congress-legislators](https://unitedstates.github.io/congress-legislators/legislators-current.json) |
| Presidential margins (House districts) | [The Downballot](https://www.the-downballot.com/p/data) Google Sheets |

| `states.topo.json` | [us-atlas](https://github.com/topojson/us-atlas) |
| `polls.json` | Phase 2 stub (empty) |

## Optional CSV overrides

Drop files in `scripts/input/` before building (used instead of MIT/MEDSL fetch):

**`house-results-2024.csv`**

```csv
district,demPct,repPct
PA-07,49.5,50.5
```

**`senate-results-2024.csv`**

```csv
state,demPct,repPct,class
PA,48.6,48.8,1
```

Columns `partyMargin`, `winner`, and `party` are optional.

**`1976-2024-house.tab`** — place the MIT house tab here to avoid re-downloading (~4 MB).

The build fails loudly if House 2024 margins are missing or incomplete (&lt;400 districts).
